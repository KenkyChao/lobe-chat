# 企业员工模型选择指南

这份手册帮你在 NaiYun AI 里快速选对模型。你不需要理解底层技术，只要先想清楚 “这次想完成什么”，再按场景挑选即可。

本文中的模型名称都以界面下拉框中的显示名为准。正式文档只公开能力范围和使用建议，不包含密钥、访问地址等敏感信息。

<Callout type={'info'}>
  公司已部署一批本地模型。标记为 “本地” 的模型免费、稳定，日常请优先使用；标记为 “云端” 的模型按需使用即可，其中旗舰模型成本更高，适合本地模型无法满足时再启用。
</Callout>

## 懒人推荐

- **日常聊天、写东西、问问题**：`DeepSeek-V4-Flash`
- **看图、看截图、看图表**：`Qwen3.6-35B-A3B`
- **复杂分析、正式输出、重要任务**：`DeepSeek-V4-Pro`
- **生成图片、做 PPT 配图**：`GPT 5.4 Image 2`、`Gemini 3.1 Flash Image Preview`
- **生成短视频、演示素材**：`Grok Imagine Video`、`Veo 3.1 Fast`
- **最难的任务、要最高质量结果**：`claude-opus-4.8`

## 使用小贴士

- **优先用本地模型**：本地模型免费、稳定，日常大多数工作场景都能覆盖
- **云端模型按需使用**：云端模型适合本地模型不够用、或者需要更高质量时再启用
- **每日 $20 消费上限**：本地模型免费不计入；云端模型按量计费，旗舰模型成本明显更高
- **看图要选支持读图的模型**：如果你要分析截图、报表、照片，请优先选择带 `读图` 能力的模型
- **图片模型不等于聊天模型**：如果你的目标是画图或配图，优先切到图片模型，而不是继续用普通聊天模型
- **出图目前走云端模型**：需要配图、草图或视觉素材时，使用 `GPT 5.4 Image 2`、`Gemini 3.1 Flash Image Preview` 等图片模型
- **视频生成走专门的视频模型**：例如 `Grok Imagine Video`、`Veo 3.1 Fast`、`Kling v3.0 Pro`
- **语音识别默认已经切到 `Qwen3-ASR-1.7B`**：日常语音输入优先直接用系统默认配置
- **长文档阅读优先选超长模型**：例如 `DeepSeek-V4-Flash`、`DeepSeek-V4-Pro`、`qwen3.5-flash`
- **简单任务不必上旗舰**：快模型通常更省时、省成本，复杂任务再升级模型即可

## 推荐组合

- **纯办公 / 写作**：`DeepSeek-V4-Flash` + `Qwen3.6-27B`，需要更强正式稿时再用 `claude-sonnet-4.6`
- **纯代码**：`DeepSeek-V4-Pro` + `qwen3-coder-next` + `minimax-m2.7`
- **低成本方案**：优先只用本地模型，已经能覆盖大多数员工场景

## 需要其他模型怎么办

如果现有模型无法满足你的实际工作需求，可以按内部流程提交申请。建议在申请时说明：

- 模型名称
- 使用场景
- 为什么当前模型不够用
- 预期使用频率

这样更方便后续统一评估和补充。
